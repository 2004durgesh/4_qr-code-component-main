# QR Code Component

This project showcases a simple webpage with a QR code component. The QR code can be scanned to visit the Frontend Mentor website, where you can enhance your front-end coding skills.

## Introduction

The QR Code Component project provides a visually appealing way to encourage users to explore and improve their front-end skills. By scanning the QR code, users are directed to the Frontend Mentor website, where they can access various coding challenges and resources to enhance their coding abilities.

## Features

- Display a QR code that can be scanned
- Link to the Frontend Mentor website
- Responsive design for different devices

## Technologies

The project is built using the following technologies:

- HTML
- CSS
- QR code image (e.g., `image-qr-code.png`)
- Google Fonts (Outfit font)


## Usage

1. Open the webpage in a web browser.
2. Scan the displayed QR code using a QR code scanner app on your smartphone or tablet.
3. You will be redirected to the Frontend Mentor website, where you can explore coding challenges and resources to enhance your front-end skills.
